==============================================
IMPORTANT NOTES FOR GITHUB PAGES
==============================================

1. All image paths in HTML files are RELATIVE to the page location
   - index.html uses: assets/img/wilson1.jpg
   - project pages use: ../assets/img/wilson1.jpg

2. Image paths in JSON config are relative to the site root
   - site-config.json uses: assets/img/wilson1.jpg

3. After pushing changes to GitHub:
   - GitHub Pages will rebuild the site automatically
   - New images will appear within 1-2 minutes
   - Hard refresh your browser (Ctrl+F5 or Cmd+Shift+R) to see changes

4. Image files MUST be committed to the repository
   - Images stored in browser localStorage (from old edit mode) will NOT persist
   - Only files in the repo will appear on GitHub Pages

5. Supported formats: JPG, PNG, GIF, SVG, WebP
