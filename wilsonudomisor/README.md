# Wilson Sunday Udomisor - Portfolio

**Data Engineer and Data Scientist**

This is the unified portfolio website showcasing work across data engineering and data science.

## Quick Start

To view the site locally:

```bash
cd wilsonudomisor
python -m http.server 8000
```

Then open your browser to `http://localhost:8000`

## Structure

- `index.html` - Landing page
- `about.html` - About page with skills and approach
- `projects.html` - All projects with category filters
- `projects/` - Individual project case study pages
- `assets/data/` - Site configuration and project data (JSON)
- `assets/css/` - Styles
- `assets/js/` - JavaScript for rendering and interactions
- `assets/img/` - Images (wilson1.jpg, wilson2.jpg)
- `docs/` - Documentation including image map, LinkedIn profile, and resume

## Key Features

- Responsive design with modern dark theme
- Dynamic project loading from JSON
- Clean separation between data engineering and data science projects
- Consistent branding across all pages
- Social links: LinkedIn, TikTok, YouTube, Email
